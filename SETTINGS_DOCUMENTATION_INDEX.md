## 💡 Key Concepts to Understand

### Backend-Driven Architecture
All data comes from database, not calculated in frontend.
**Why?** Accuracy, consistency, single source of truth.

### RPC (Remote Procedure Call)
A function in the database that frontend can call.
**Why?** Efficient, returns all data in one query.

### Single Source of Truth
One place where data lives and is retrieved from.
**Why?** No inconsistencies, auditable, reliable.

### Real-Time Data
Data fetched fresh every time, not cached.
**Why?** Always current, no sync issues.
